  • TYPES OF SEO

  • ON Page SEO :- Can be done by HTML developers
  • OFF Page SEO :- Can be done when i will put another website link in my own Page or website.
  • On-page SEO includes providing good content, good keyword selection, putting keywords in correct places, giving an appropriate title for every page,etc.
  • Off-page SEO includes link building, increasing link popularity, search engine, link exchange etc. 2. On-page SEO will analyze the complete website.
  • HTML developers can implement SEO using the following Techniques :-
  1. Set good title & to the point
  2. Set the meta description
  3. Set a nice URL slug
  4. Set the meta author tag
    5. meta name="author" context="gg" :- for writing blog for which author and yea user like it
  5. Set a favicon
    6. What is Fav icon?

    A favicon is a small file containing the one or more icons which are used to represent the website or a blog. It is also known as a tab icon, website icon, URL icon, or a bookmark icon.

    ctrl+shift+r for icon showing in upper URL.
